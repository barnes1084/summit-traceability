# Summit Traceability Platform

Hybrid local + cloud traceability and inventory system for small-to-mid manufacturers.

## Architecture

- ASP.NET Core backend
- Angular PWA frontend
- PostgreSQL (Dockerized)
- Vendor-neutral scan ingestion layer
- Optional Android Intent bridge for rugged scanners

## Goals

- Simple scan-based traceability
- Offline-capable edge deployment
- Modular expansion (machine data, cloud sync)
- Vendor-neutral scanner support
